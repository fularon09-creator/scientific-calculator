# 🧮 SciCalc Pro — Advanced Scientific Calculator

**Live Demo:https://science-calc-pro.replit.app

---

## Overview

SciCalc Pro is a sleek, fully responsive **Scientific Calculator** built for 
students, engineers, and professionals who need powerful computation at their 
fingertips — right from their browser. No downloads. No installs. Just results.

---

## ⚡ Features

- **Full Scientific Functions** — sin, cos, tan, log, ln, π, e, and more
- **Memory Operations** — MC, MR, M+, M−, MS for smooth calculations
- **Advanced Math** — x², x³, xʸ, √x, ⁿ√x, 1/x, |x|, n!, EXP
- **DEG / RAD Toggle** — seamlessly switch between degree and radian modes
- **INV Functions** — inverse trigonometric and scientific operations
- **Minimal Dark UI** — distraction-free, eye-friendly design built for focus
- **Responsive Design** — works beautifully on both desktop and mobile
- **Lightning Fast** — instant calculations with zero lag
